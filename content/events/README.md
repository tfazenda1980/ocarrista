# Eventos O Carrista

Cada evento tem um ficheiro JSON e uma rota em `/eventos/[slug]`.

## Adicionar um novo evento

1. Copie `workshop-2026.json` para `meu-evento.json`
2. Registe o slug em `app/lib/events/load-event.ts`
3. Adicione cartão na secção Eventos (opcional) e imagens em `public/images/`

## Workshop 2026

- Imagem: `public/images/workshop26.png` (fundo transparente recomendado)
- URL: `/eventos/workshop-2026`
- Teaser na homepage: `#workshop-2026`
