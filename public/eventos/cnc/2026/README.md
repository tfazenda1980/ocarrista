# Documentos CNC 2026

Coloque os PDFs nesta pasta e actualize `content/events/cnc/2026.json` com os caminhos públicos.

Exemplos:

- `/eventos/cnc/2026/nota-abertura.pdf`
- `/eventos/cnc/2026/programa-geral.pdf`
- `/eventos/cnc/2026/iniciacao/ordens.pdf`
- `/eventos/cnc/2026/galeria-premios.pdf`

No JSON, a galeria usa `galleryPdf.href` na disciplina `premios`.
